# 🍉 Watermelon Game UI Edition

참고 이미지의 분위기를 살려 만든 정적 수박 게임입니다.

## 실행
- GitHub Pages로 바로 배포 가능
- `index.html`, `styles.css`, `game.js`만 업로드하면 동작

## 조작
- 마우스 이동 / 터치 이동: 떨어뜨릴 위치 조절
- 클릭 / 터치 / Enter / Space: 과일 떨어뜨리기
- 다시하기 버튼으로 재시작

## 특징
- 스코어 / 최고 점수 버블 UI
- 다음 과일 표시
- 좌우 장식 패널 포함
- 모바일 대응
- 로컬 최고 점수 저장
