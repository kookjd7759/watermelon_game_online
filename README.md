## 🚀 Deploy

레포를 **Public**으로 만든 뒤 아래 순서로 배포하세요.

1. `Settings`
2. `Pages`
3. `Deploy from a branch`
4. Branch: `main` / Folder: `/root`
5. 저장 후 몇 분 뒤 Pages 주소가 활성화됩니다.

## 🕹️ Controls

- **마우스 이동 / ← →** : 과일 위치 이동
- **클릭 / Enter / Space** : 과일 떨어뜨리기
- **R** : 다시 시작
