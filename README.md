# 🍉 Watermelon Game HQ

GitHub Pages에 바로 올려서 실행할 수 있는 고퀄리티 수박 게임입니다.

## 실행
`index.html`, `styles.css`, `game.js` 그대로 업로드한 뒤 GitHub Pages를 켜면 됩니다.

예상 주소:
`https://kookjd7759.github.io/<repo-name>/`

## 특징
- Canvas 기반 가벼운 렌더링
- 모바일 / 데스크톱 입력 지원
- 합체 파티클, 화면 흔들림, 점수 / 최고기록
- localStorage 기반 최고점 저장
- 단일 정적 파일 구조라 Pages 배포가 쉬움

## 권장 레포 이름
- `watermelon-game`
- `watermelon-game-hq`
- `suika-game-kr`
- `fruit-merge-game`

## Pages 배포
1. 이 폴더의 파일을 레포 루트에 업로드
2. 저장소를 **Public** 으로 변경
3. `Settings -> Pages`
4. `Deploy from a branch`
5. `main` / `/root`
6. 저장 후 배포 주소 접속
